# frontend-boilerplate

Front-end boilerplate to save your time from kicking off new projects.

## Features

- Pug, SCSS, JS/ES6 supported
- Live reload
- Auto add CSS prefix
- Clean unused CSS
- Minify images
- Concat and minify code in production

## Usage

### Install dependencies

```
npm install
```

### Run local server

```
npm run dev
```

### Build for production

```
npm run build
```
